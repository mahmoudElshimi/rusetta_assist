Welcome to the Rusetta Assist Odoo Module!

A tool for managing student attendance, exams, and marks.

Manage Student Attendance and Exam Records

Track and manage educational data efficiently

Record student attendance and track their last login status.

Manage and assign marks for various exams.

Filter and group data based on teachers, exams, dates, and marks.

Installation Instructions

How to get started with Rusetta Assist

To install the Rusetta Assist module, you can choose one of the following methods:

Clone the Repository:

Clone the rusetta_assist directory into your custom addons path: 

git clone ssh://git@github.com/mahmoudElshimi/rusetta_assist.git /path/to/your/custom/addons/rusetta_assist

Odoo Apps Store:

Alternatively, you can install the module directly from the Odoo Apps Store:

[Rusetta Assist on Odoo Apps](https://apps.odoo.com/apps/modules/17.0/rusetta_assist)
